## iCollections 是什么？

简单说，它就是「Mac 版的 Fences」——Windows 用户应该都知道 Fences 那个把图标圈进分区的功能，iCollections 在 Mac 上把这件事做得更彻底，还加了一堆额外的实用功能。

核心逻辑很直接：在桌面上划出若干「集合区域」，把文件、文件夹、应用拖进去分类摆放。不用时区域自动折叠，鼠标靠近就弹出来。整个桌面从「垃圾场」变成「可视化文件柜」。

## 主要功能拆开讲

**桌面分区整理**

这是核心功能。新建一个集合，拖进文件，完事。你可以给每个区域命名、调整尺寸、换颜色，还能设置折叠行为——不用时缩成一个小标签，点一下才展开，桌面照样清爽。

**文件中转站**

批量移动文件时特别好用。假设你要把十几个文件分别移到不同文件夹，以前要一个个拖，现在先全部扔进中转站，再慢慢分配，效率高了不少。

**网页视图**

直接把网站嵌进桌面——常用的任务管理工具、日历、实时汇率，不用开浏览器，瞄一眼桌面就行了。有点「桌面小组件」的感觉，但更灵活。

**相框 & 幻灯片**

把照片以幻灯片形式显示在桌面上，每隔一段时间自动切换。工作累了抬头看一眼家人的照片，也挺治愈的。

**窗口 & 应用监视器**

桌面上挂一个应用进程管理器，快速切换应用和窗口，比 ⌘Tab 更直观，还能设置「触发角」来快速操作窗口。
